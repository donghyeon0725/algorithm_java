자료구조
-

자료구조를 사용하는 목적
-
대량의 데이터를 효율적으로 관리하기 위함 (찾고 입력할 때)

ex) 배열, 스택, 큐, 링크드 리스트, 해쉬테이블 등등

알고리즘을 배우는 목적
-
알고리즘이란?
   - 문제 풀이를 위한 절차/방법 
   - 입력을 받아 출력하는 과정을 조절하는 것

ex) 요리 레시피

좋은 알고리즘이란?  
   - 시간, 작은 공간(메모리)을 차지하는 알고리즘
